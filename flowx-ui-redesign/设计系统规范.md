# FlowX 界面设计规范（功能对齐版 v2）

> 本规范基于 `flowx-desktop` 真实源码（App.vue / 7 个页面 / router / types）整理，确保设计与已实现功能 100% 一致。
> 配套高保真原型：`index.html`（浏览器直接打开即可逐页预览与交互）。

---

## 一、品牌与视觉基调（对齐现有代码）

FlowX 现有代码的真实品牌是 **靛蓝 → 天蓝渐变 + 磨砂玻璃**，并非通用"紫蓝 AI 渐变 slop"，也非奶油暖色。所有主色、激活态、Logo 均取自源码：

| Token | 值 | 用途 |
|---|---|---|
| `--indigo` | `#6366F1` | 主品牌色（激活态/主按钮/选中） |
| `--sky` | `#38BDF8` | 次品牌色（渐变收尾/信息） |
| Logo 渐变 | `linear-gradient(135deg,#6366F1,#38BDF8)` | 侧栏 Logo（与 `App.vue` 一致） |
| 侧栏背景 | `rgba(255,255,255,.72)` + `backdrop-filter: blur(30px)` | 磨砂玻璃 |
| 面板 | `#fff` / `rgba(255,255,255,.9)` + 1px 描边 + 柔和阴影 | glass 卡片 |

**平台语义色**（仅小面积用于徽标/状态，不与主色冲突）：

| 平台 | 色值 | 视频 | 图文 | 文章 |
|---|---|---|---|---|
| 抖音 | `#06B6D4` | ✅ | ✅ | ✅ |
| 小红书 | `#FF4D4F` | ✅ | ✅ | ✅ |
| 快手 | `#F97316` | ✅ | ✅ | ❌ |
| 微信视频号 | `#10B981` | ✅ | ✅ | ❌ |

> 注：架构层 `KNOWN_PLATFORMS` 还含 zhihu/toutiao 等占位，但 README 的"已支持平台"以抖音/小红书/快手/微信视频号为准。Dashboard 按实际已注册平台动态渲染。

**状态色**：成功 `#10B981` / 警告 `#F59E0B` / 危险 `#F43F5E` / 信息 `#64748B`。

## 二、导航结构（真实路由）

```
仪表盘        /dashboard
账号管理      /accounts
一键发布      /publish
发布历史      /history
系统配置      /settings  （子菜单，非独立页）
  ├ 环境配置   /settings/environments
  ├ 代理 IP 设置 /settings/proxies
  └ 对外接口   /settings/api
```

> ⚠️ 修正点：原方案误加了"数据中心"页（实为路线图 v1.2，未实现），并把"系统配置"做成泛化单页。**真实结构是上述 7 个页面 + 系统配置子菜单**。

## 三、页面功能清单（已实现，须逐项覆盖）

### 1. 仪表盘 Dashboard
- 4 张指标卡：已授权账号 / 活跃账号 / 历史总发布 / 今日成功数
- 快捷导航：管理账号 / 一键发布 / 浏览历史
- 支持平台生态：各平台卡片显示"已绑定 X 个账号"

### 2. 账号管理 AccountPanel（功能丰富，原方案遗漏较多）
- 顶部：自动检测状态标签（启用/间隔）
- 操作：授权新账号 / 分类管理 / 定时设置（健康检测配置）/ 批量检测健康 / 刷新
- 按分类筛选
- 账号卡片：平台徽标 + 状态灯（正常/已过期/未激活）、头像、昵称、备注、平台号、粉丝/关注/获赞、分类标签、绑定环境标签（或"本机直连"）、授权/检测时间
- 卡片操作：创作中心 / 编辑 / 刷新 / 删除
- 弹窗：选择平台授权（平台选择 + 绑定浏览器环境）、编辑账号（昵称/备注/分类≤5/环境）、分类管理（增删改）、定时检测设置（启用/间隔/首次延迟）

### 3. 一键发布 Publish
- ① 发布类型：视频 / 图文 / 文章
- ② 标题与内容：标题（按平台动态字数上限）、素材（图片/视频格式限制）、封面（文章模式抖音必填 / 视频可选）、描述（动态字数 + 快手500字/超限/文章最小字数提示）、话题
- ③ 选择账号：按分类筛选 + 全选可见/清空 + 账号网格（按 capability 过滤，显示平台/昵称/字数限制）
- ④ 发布设置：立即发布 / 定时发布（datetime picker）
- 提交 + 调试模式
- 任务进行中：折叠任务、总进度、逐账号状态/进度/详情/结果链接、移除
- 调试日志终端（实时）

### 4. 发布历史 History
- 卡片列表：ID / 内容类型标签 / 状态徽章 / 创建+定时时间 / 标题 / 内容预览 / 话题 / 分发账号结果徽章（✓/✗）
- 操作：查看详情 / 取消发布（scheduled）/ 重试（有失败）/ 编辑重发（有失败）/ 删除
- 分页
- 详情弹窗：元数据网格 + 逐账号执行明细表

### 5. 环境配置 BrowserEnvPanel
- 环境卡片：名称 / 已激活 / UA 预览 / 关联代理 / 创建时间 / 编辑 / 删除
- 弹窗：名称 / User-Agent（textarea + **随机生成**，内置 5 套模板）/ 绑定代理 IP

### 6. 代理 IP 设置 ProxyPanel
- 代理卡片：名称 / 协议标签 HTTP·SOCKS5 / 服务器地址 / 用户名 / **代理状态**（测试→可用+延迟+出口IP）/ 创建时间 / 测试连接 / 编辑 / 删除
- 弹窗：名称 / 协议 / 主机 / 端口 / 用户名 / 密码

### 7. 对外接口 ApiPanel
- 服务运行状态标签
- 服务配置：启用开关 / 监听端口 / API Key / 保存 / 启动 / 停止
- 接口文档（可展开）：`GET /api/health`、`GET /api/accounts`、`POST /api/publish`、`GET /api/publish/:taskId`，含参数表与 JSON 示例
- 注意事项

## 四、组件规范要点

- **侧栏菜单项**：全圆角胶囊（radius 24px），图标外层圆形容器；hover/active 用靛蓝 tint + 柔光阴影。
- **主按钮**：`linear-gradient(135deg,#6366F1,#38BDF8)`，白字，radius 12px。
- **卡片**：白底 + 1px 描边 + 柔和阴影，radius 14~18px，hover 轻微上浮。
- **表格/标签**：Element Plus 风格但去后台感，圆角 + 同色系 tint。
- **动效**：`cubic-bezier(.16,1,.3,1)`；页面切换 fade+位移；状态变化即时可见。

## 五、无障碍与落地

- 对比度满足 WCAG AA（深墨文字 `#0F172A` on 浅底）。
- 交互元素最小尺寸 ≥ 36px。
- 组件名与现有 `*.vue` 文件一致，便于直接替换或主题化落地（现有项目已用 Element Plus + 自建磨砂样式）。
