顶栏全局任务选项卡
中部弹性区域承载任务选项卡，与右侧搜索栏保持恒定间距；开再多标签也不挤压搜索栏。
当前激活任务：—
（此处为对应任务的内容区，原型仅示意）
设计说明 / 交付规范
供开发落地的关键参数（可直接转 Element Plus / Vue 组件）。
区域定义
顶栏中部设弹性容器 flex:1; min-width:0，专属承载全局任务选项卡；永不挤压左右元素。
与搜索栏间距
固定 18px 间隔 ×2 + 1px 分隔线（约 37px），恒定不变，不随标签数量变化。
溢出策略
标签超出可视宽度时，区域内横向滚动，左右 ◀ ▶ 翻页按钮仅在溢出时出现；搜索栏位置与间距恒定。
标签规格
高 32 / 圆角 9 / 最大宽 188（超长标题省略号）/ 结构：平台色点 + 标题 + 关闭 ×。
激活态
靛蓝渐变底 + 靛蓝描边 + 柔光阴影；关闭按钮 hover 变红。
新增入口
区域右端常驻 ＋ 新建任务（虚线按钮），点击追加标签用于演示溢出。
响应式
≤1080px 搜索栏收窄至 240px；≤760px 标题文字隐藏、搜索栏 180px（建议更窄时折叠为图标按钮）。
※ 原型交互：点击标签切换激活、× 关闭、＋/压力测试 增加标签以观察溢出与恒定间距。键盘可 Tab 聚焦标签，符合 WCAG 可达性。
